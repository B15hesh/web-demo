Mini Personal Portfolio Website

This website is designed and developed as part of Web Technology Lab #2. The main objective of this project is to demonstrate the practical
use of HTML elements including text formatting, lists, tables, embedded media, iframes, and basic HTML events.

The entire website is created using HTML only, without using CSS or JavaScript, as per the lab requirements.
