Academic Projects

This page presents a summary of the academic projects completed as part of my coursework during the BSc CSIT program. These projects helped
me understand problem-solving, logical thinking, and web fundamentals.

List of Academic Projects

  1. Mini Personal Portfolio Website

    This project involves designing a multi-page personal portfolio website using HTML only. It demonstrates navigation, semantic layout, embedded
    media, iframes, and HTML events.

    • Technology Used: HTML
    • Type: Individual Project
    • Purpose: Web Technology Lab
  2. Online Quiz System

    The Online Quiz System is a conceptual academic project designed to conduct quizzes with multiple-choice questions. It focuses on user interaction
    and question management.

    • Technology Used: HTML (Conceptual)
    • Type: Group Project
    • Purpose: Academic Practice
  3. Student Management System

    This project is aimed at managing student information such as personal details, academic records, and results. It provides a basic understanding of
    system design.

    • Technology Used: HTML (Conceptual)
    • Type: Academic Project
    • Purpose: System Design Understanding
